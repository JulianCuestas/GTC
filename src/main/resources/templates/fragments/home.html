Bienvenido a GTC

Sistema de Gestión de Tarjetas de Crédito.

Caracteristicas del proyecto:
  • Lenguaje: Java 1.8
  • SGBD: PostgreSQL
  • PaaS: Heroku
  • Frameworks: SpringBoot
  • Patrones: MVC, Singleton, Factory
  • Libs Front: Thymeleaf, Bootstrap, Datatable, jQuery, SweetAlert
